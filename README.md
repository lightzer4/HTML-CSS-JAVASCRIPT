HTML, CSS & JavaScript - Estudos e Projetos

Este repositório foi criado para centralizar e organizar meus estudos sobre as tecnologias fundamentais da web (HTML5, CSS3 e JavaScript). Aqui você encontrará projetos práticos, exercícios de lógica e interfaces desenvolvidas durante o meu aprendizado de front-end.
🚀 Tecnologias Utilizadas

    HTML5: Estruturação semântica de conteúdo.

    CSS3: Estilização, layouts (Flexbox/Grid) e animações.

    JavaScript (ES6+): Manipulação de DOM, eventos e lógica de programação.
